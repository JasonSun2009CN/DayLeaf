## 中文版 README.md

# DayLeaf

一款运行在 macOS 平台 的轻量日记记事应用，专注简洁体验，支持标签分类与心情标记。

简介

DayLeaf 是一款简洁、自然的 macOS 日记应用，以日历视图为核心，帮助你轻松记录日常心情、灵感与生活片段，支持标签管理和表情心情标记，让日记分类更清晰、记录更有温度。

# 核心功能

- 📅 直观的日历展示，快速查看与跳转日期，一目了然掌握每日记录情况

- ✏️ 简洁流畅的记事编辑体验，专注内容创作，无多余干扰

- 🏷️ 日记标签（tag）分类功能，可自定义标签，轻松归类、检索日记内容

- 😀 心情（mood）标记功能，用 emoji 直观表达当日心情

- ⚡ 轻量化设计，占用资源少，启动快速，专注日记本身

# 开发与生成

- 整体采用 vibe coding 风格开发

- 由 ChatGPT 生成提示词与 ASCII 图像

- 代码由 QClaw 生成（默认使用腾讯大模型）

- 应用图标由 Doubao（豆包） 生成

# 平台

- macOS

# 协议

开源协议：MIT

## English README.md

# DayLeaf

A lightweight diary & note-taking application for the macOS platform, focusing on a clean experience with tag classification and mood marking.

Introduction

DayLeaf is a simple and natural diary app built for macOS. Centered around a calendar view, it helps you easily record daily thoughts, moods, inspirations, and life moments. It supports tag management and emoji mood marking, making diary classification clearer and recording more heartfelt.

# General Features

- 📅 Intuitive calendar presentation for quick date navigation and review, easily keeping track of daily records

- ✏️ Clean and smooth note-editing experience, focusing on content creation without unnecessary distractions

- 🏷️ Diary tag classification function, allowing custom tags for easy categorization and retrieval of diary content

- 😀 Mood marking function, using emojis to intuitively express the mood of the day

- ⚡ Lightweight design with low resource usage, fast startup, and focus on the diary itself

# Development & Generation

- Developed with vibe coding

- Prompts and ASCII images generated by ChatGPT

- Code generated by QClaw (default LLM from Tencent)

- App icon generated by Doubao

- macOS
